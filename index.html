Alzheimer’s Disease Detection using Computer Vision and Machine Learning Techniques

• Aimed towards a multifaceted approach combining cognitive analysis, CV, and ML for early detection of Alzheimer’s disease. Collecting and preprocessing cognitive data (Handwriting, Facial Expression, Gait Analysis) and brain imaging analysis, extracting key features.

Automated Hexcode and RGB Code Detection from images

An automated system was designed for detecting Hexcode and RGB colour codes from images (live or static) and developed algorithms to extract colour information. These algorithms were integrated into a robust colour detection tool.

Development of Polymer based Pressure Sensors and Enhanced Material Expansion Detection using Machine Learning

• Custom polymer-based piezoresistive sensors were fabricated using PEDOT:PSS and PDMS, utilising 3D printing for microstructure fabrication. The different microstructure dimensions and shapes on the polymer and their response to pressure variations were studied.
• Machine learning techniques for predictive analysis and testing the sensors on inflatable objects with variable pressure were employed to achieve optimised results.
